# 老宅延壽機能復新計畫 — 修繕補助勾選單

> 透天住宅修繕補助勾選工具，依內政部國土管理署「老宅延壽機能復新計畫」整理

🌐 **線上使用：[a5347393.github.io/renew-app](https://a5347393.github.io/renew-app/)**

---

## 功能介紹

### 勾選補助項目
依照申請流程分為三大區塊：
- **前置評估**：初步或詳細結構安全性能評估（二選一）
- **室外修繕**：立面、屋頂、空調管線、無障礙設施（至少選 1 項）
- **室內修繕**：居家安全、管線更新、室內裝修（需先完成至少 1 項室外）

### 展開細項勾選
每個主項目可展開，逐一勾選預計施作的具體工程，支援：
- **全選** / **全取消** 快速操作
- 即時顯示已選細項數量（例如 `3/7`）
- 細項卡片勾選後變色，一眼辨識

### 備註給設計師
每個項目有獨立備註欄，並提供對應的填寫範例：
- 立面修繕 → 「2、3 樓北側外牆磁磚脫落嚴重需優先處理...」
- 屋頂工程 → 「屋頂有積水問題，排水坡度需重新整坡...」
- 管線更新 → 「全棟給水管為舊式鍍鋅鐵管，電箱容量不足需升級至 50A」
- …等 9 組對應範例

### 樓層選擇
勾選「建築物立面修繕」後，可選擇樓層數自動套用對應補助上限：
- 1–2 層 → 100 萬
- 3–4 層 → 200 萬
- 5–6 層 → 300 萬

### 加碼補助條件
居家安全項目支援加碼 30 萬/戶，適用條件：
- 65 歲以上長者
- 低收入或中低收入戶
- 長照需要等級第 2 級以上

### 即時補助上限試算
底部 Sticky 列即時加總所有勾選項目的補助上限，清楚標示每項補助上限為實際工程費 × 65%。

### 匯出 PDF（手機電腦皆支援）
點選右上角「⬇ 匯出 PDF」，使用 html2canvas + jsPDF 在瀏覽器內直接產生 PDF 並下載，內容包含：
- 業主姓名、房屋地址、設計師、日期
- 所有勾選項目及補助上限
- 已勾選的施作細項
- 各項備註
- 補助上限合計

## 申請規則

| 項目 | 說明 |
|------|------|
| 申請資格 | 30 年以上合法透天住宅，完成結構安全性能評估 |
| 補助比例 | 各項最多補助實際費用 **65%** |
| 室內修繕前提 | 需先完成至少 **1 項室外修繕** |
| 補助對象 | 非法人，住宅使用 |

## 本地開發

```bash
npm install
npm run dev
```

開啟 http://localhost:5173 預覽。

## 部署

推送到 `main` 分支後，GitHub Actions 自動建置並部署到 GitHub Pages。

```bash
git add .
git commit -m "update"
git push
```

## 技術架構

- **React 18** + **Vite 5**
- **html2canvas** + **jsPDF** — 瀏覽器端 PDF 產生（支援手機）
- 無後端、無資料庫，純前端靜態網頁
- GitHub Actions 自動部署
